# Декодирование AIS сигнала в [GRC](https://www.gnuradio.org/) на базе стандартных компонент (блоков)

## Постановка задачи 
1. Разобраться в принципах работы [GRC](https://www.gnuradio.org/) (на начало 2026 в РФ сайт не работает без VPN)
2. Провести эксперименты с демодуляцией GMSK
3. Провести извлечение пакетов AIS без дополнительного кода

## Окружение 
### Софт
* Операционная система Ubuntu Ubuntu 24.04.3 LTS
* GNU Radio v3.10.9.2 (Python 3.12.3)
